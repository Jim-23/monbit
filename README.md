# Monbit
A tiny ESP32-based digital monster pet with an e-ink display that evolves over time based on care, sleep cycles, and randomness.


## Monbit roadmap

### 0.0.1 — “Hello hardware”

Goal:
- ESP32 boots
- serial prints work

Hardware:
- ESP32-C3

Done when:
- you can upload code reliably


### 0.0.2 — “Display alive”

Goal:
- 1.54" SPI E-Ink Module shows image/text

Done when:
- first monster is born on the screen

### 0.0.3 — “Input system”

Goal:
- buttons are buttoning

Hardware:
- 3 buttons wired

Done when:
- press button → state changes in serial/log
